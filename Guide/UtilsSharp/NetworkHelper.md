```c#
/// 是否能Ping通指定主机
/// </summary>
static bool Ping(string ip)
```

#### 2、网络是否畅通：IsInternetConnected

```c#
/// <summary>
/// 网络是否畅通
/// </summary>
static bool IsInternetConnected()
```
